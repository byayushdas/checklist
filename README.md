# Soroban Checklist DApp

## Project Description
This project is a decentralized application built on the Stellar network using Soroban smart contracts. It provides a simple, on-chain checklist or "to-do list" application, demonstrating the fundamental features of Soroban smart contract development: state management and custom complex types.

## What it does
The Checklist Smart Contract allows you to:
1. Create new tasks, which are securely stored on the blockchain.
2. Mark existing tasks as completed.
3. Retrieve task information by its ID.

This forms a solid baseline for any application tracking completion states on-chain, such as milestones, voting tallies, or supply chain stages.

## Features
- **Add Task:** Start tracking an item by submitting a string description.
- **Mark Done:** Irreversibly mark a task as completed on-chain.
- **Get Task:** Read specific task details (description and completion status).
- **Persistent State:** All tasks are recorded and retrieved from the Soroban contract instance storage.
